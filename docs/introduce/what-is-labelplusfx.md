# LabelPlusFX是什么?

LabelplusFX是一个基于原版[LabelPlus](https://github.com/LabelPlus/LabelPlus)的设计理念，使用JavaFX进行开发的漫画汉化辅助工具，支持win/macOS双平台。其相对于原版做出了一定的改进，并引入了一些新特性，带来了更好的使用体验。

## 改进

- 图片可以在窗口中任意拖动
- Label与提示文本显示优化
- 翻译和标号模式提示更清楚
- 支持多种图像格式（BMP、GIF、PNG、JPG、JPEG、JFIF、TIF、TIFF、WEBP）
- 更多分组
- 可以手动指定图片文件路径
- 等等

## 特性

- 提供JSON格式的Meo版翻译文件
- 更方便的分组管理
- 可拖动的Label
- 等等

## 内置工具

- 中日小词典
- 繁简转换
- 文件即时备份与恢复
- 提供文本格式检查功能
- 等等
